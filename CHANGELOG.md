# CloudPilot 云驾 - 更新日志

## v1.2.0 (2026-07-18)

### 🎯 SRE 工作流深度优化

**核心场景重构** - 从功能导向改为状态+任务流导向：

- **底部 Tab 重构**：总览 / 告警 / 诊断 / 任务 / 更多
- **首页仪表盘**：SRE 状态条（集群健康、待处理告警、今日任务数）
- **诊断中心**：日志 + 监控统一入口，未来支持联动分析
- **任务中心**：异步操作进度看板，进行中任务自动刷新
- **告警工作台**：聚合展示、一键操作（确认/静音/查日志/查监控）

### ✨ 新功能

- **全局搜索**：浮动搜索按钮，支持集群/Pod/命名空间/快捷命令模糊匹配
- **Pod 详情操控中心**：顶部固定操作栏（监控/日志/终端/下载/重启）
- **触觉反馈**：重要操作带震动反馈（重启/删除/成功/失败）
- **持续运营**：更新日志、问题反馈、GitHub 项目入口

### 🐛 修复

- **OTA 更新死机**：修复 setServerBasePath 导致的 WebView 死锁
- **启动异常循环**：新增健康检查，10 秒内重启会自动清除 OTA 数据回退到 builtin 版本
- **网络不通提示**：从大卡片改为顶部小横幅
- **CLS 日志时间戳**：从 UnixSecond 改为 UnixMilli
- **CLS 日志搜索吞字**：改用独立 input state 避免防抖丢字

### 📱 已知问题

- 服务器 SSH 连接偶发超时（EIP 变更导致，重试即可）
- APK 编译需 5-8 分钟（GitHub Actions 排队时间）
