## 决策 Sign-off（已确认）

- **主场景**：飞书告警驱动 70% + 主动巡检 30%
- **使用环境**：80% 走路/地铁（单手），20% 坐下细看
- **决策模式**：AI 建议为主 + 自己看图为辅（AI 建议是 V2）
- **数据延迟**：<30s，2 分钟不可接受
- **数据源**：Grafana 主 + Prometheus 交叉 + kubectl 校验，不一致时并列展示
- **预检查**：配额 / 节点状态 / 5 分钟去重 / 成本估算 / 强制节点池
- **UI 风格**：飞书商务简洁 + Grafana 深色数据
- **操作步骤**：5-7 步，但每步信息要全
- **审批流**：MVP 不做
- **前端**：Claude 全部开发（飞书小程序 TMA）
- **后端**：Go + PostgreSQL
- **部署**：10.210.20.130:/data2/haowu33/mobile 单机
