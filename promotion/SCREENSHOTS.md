# CloudPilot 云驾 - 宣传截图标准指引

## 一、截图规范

### 1.1 设备与分辨率

**推荐设备**: Android 旗舰机（MIUI/EMUI/原生 Android）

**分辨率要求**:
- 纵向: 1080 x 2400 或 1440 x 3200 (9:20 或 9:19.5 比例)
- 横向: 不推荐（运维场景主要竖屏操作）

**避免**: 刘海屏、水滴屏的异形区域（截图时隐藏或裁剪）

### 1.2 截图前准备

- ✅ 充满电（电池图标 90%+ 更美观）
- ✅ Wi-Fi 信号满格
- ✅ 清理通知栏（避免杂乱信息）
- ✅ 调整时间到 10:15 / 14:30（黄金时刻，电池图标和信号格都好看）
- ✅ 深色模式（推荐 — 科技感强）或浅色（看场景）
- ⚠️ 隐藏真实数据（IP/集群名/节点名 → 用测试数据/脱敏处理）

### 1.3 截图工具

**Android 原生**:
- 电源键 + 音量下（同时按）
- 三指下滑（部分机型）
- 下拉通知栏 → 屏幕截图

**高级工具**（可裁剪、打码、加外框）:
- **Snapseed**（免费，Google 出品）
- **PicsArt**
- **Figma/Sketch**（设计师专用 — 可批量加外框）

## 二、必拍 10 张核心截图

### 截图 #1: 集群概览
**场景**: 首页 / Clusters 页展示多集群列表
**要点**:
- 3-5 个集群卡片（测试/生产/开发环境）
- 每个卡片显示 Nodes/Pods/CPU 使用率
- 顶部搜索框 + "+" 新增按钮
- 底部 Tab 导航高亮 "Clusters"

### 截图 #2: Pod 列表（带实时状态）
**场景**: 进入某个集群 → Resources → Pods
**要点**:
- 列表显示 10+ Pods，包含 Running/Pending/CrashLoopBackOff 混合状态
- 顶部 Tabs（Pods/Deployments/Services...）高亮 Pods
- 顶部搜索栏 + 命名空间筛选（显示当前 ns）
- 右上角 "实时监听中" 绿点标记

### 截图 #3: Pod 详情 - 基础信息
**场景**: 点进某个 Pod → 基础信息 Tab
**要点**:
- 顶部 Pod 名称 + namespace
- Tabs: 基础信息 / 日志 / 监控 / 事件 / YAML / 操作
- 卡片: Labels / Annotations / 容器信息（镜像/资源限制）
- 底部操作按钮区: 重启/删除/导出 YAML

### 截图 #4: 实时日志流（滚动中）
**场景**: Pod 详情 → 日志 Tab，日志正在滚动
**要点**:
- 顶部容器选择器（多容器 Pod 场景）
- 日志区域显示带时间戳的日志行（高亮 ERROR/WARN）
- 底部操作栏: 暂停/恢复、跳到底部、下载、分享
- 状态提示: "实时流连接中 🟢" 或 "暂停"
### 截图 #5: 监控图表（CPU/内存/网络/磁盘）
**场景**: Pod 详情 → 监控 Tab
**要点**:
- 4 张 ECharts 时序图（CPU/内存/网络 Rx+Tx/磁盘 Read+Write）
- 顶部时间范围选择器（5分钟/15分钟/1小时...）高亮当前选中
- 图表显示最近数据，线条平滑，有 tooltip 悬浮
- 下方 Requests/Limits 卡片（资源配置）

### 截图 #6: 事件列表（含告警）
**场景**: Pod 详情 → 事件 Tab
**要点**:
- 顶部 "实时监听中 🟢" 状态标记
- 事件列表：Normal/Warning 混合（边框色区分）
- 每条事件显示: Type tag + Reason + Message + 时间戳 + 重复次数
- 最新事件在顶部

### 截图 #7: YAML 编辑器（修改中）
**场景**: Pod/Deployment 详情 → YAML Tab，点击"编辑"按钮进入编辑态
**要点**:
- 顶部按钮行：💾保存并应用 / 取消
- 可编辑 textarea，YAML 语法高亮（如果有）或 monospace 字体
- 显示部分 spec 内容（replicas/image/env 等关键字段）
- 底部提示："保存前会自动备份快照"

### 截图 #8: 快捷操作面板
**场景**: Deployment 详情 → 操作 Tab
**要点**:
- 顶部"工作负载操作"卡片
- 按钮组: 🔄滚动重启 / 🔢扩缩容(当前3) / ⏸️暂停 / 🗑️删除
- 每个按钮有清晰图标 + 文字
- 下方可选显示历史操作记录或灰色提示区

### 截图 #9: 告警通知（含语音播报）
**场景**: 后台收到 Prometheus 告警推送，App 前台显示通知卡片
**要点**:
- 顶部通知卡片：红色边框 + 🔔图标
- 标题: "[告警] production-cluster"
- 内容: "nginx-ingress-controller CPU 使用率 92%"
- 底部操作: "查看详情" / "忽略"
- （可选）震动 + 语音播报文字悬浮提示

### 截图 #10: 设置页（含 OTA 更新）
**场景**: 底部 Tab → 设置
**要点**:
- 顶部用户卡片（头像 + 用户名 + 角色）
- 分组 List: 外观（深色/浅色/跟随系统）/ 数据源（Grafana/Prom）/ 通知设置
- "检查更新" 菜单项（右侧显示当前版本号小字）
- 底部: 退出登录（红色）

## 三、Feature Highlights（文案 + 图文排版）

用于落地页、GitHub README、App Store 介绍页的核心卖点展示。

### Feature #1: 🚀 随时随地运维
**标题**: 深夜告警？外出办事？手机秒开集群
**描述**: 
- 支持多 K8s 集群管理（阿里云/腾讯云/自建）
- Pod/Deployment/Service/Node 一键查看
- 实时状态更新，无需刷新

**配图**: 截图 #1 集群概览 或 #2 Pod 列表

---

### Feature #2: 📊 实时监控与日志
**标题**: CPU 飙升？Pod 重启？一眼看清
**描述**:
- ECharts 时序图：CPU/内存/网络/磁盘 I/O
- 实时日志流（SSE 推送），支持暂停/搜索/下载
- 事件流监听：Normal/Warning 自动归类

**配图**: 截图 #5 监控图表 + #4 日志流（拼图）

---

### Feature #3: 🔧 快捷操作，秒级响应
**标题**: 重启？扩容？删除？点一下就行
**描述**:
- 滚动重启 Deployment/StatefulSet/DaemonSet
- 在线扩缩容（修改 replicas）
- YAML 在线编辑 + 历史版本回滚
- 安全确认弹窗，防止误操作

**配图**: 截图 #7 YAML 编辑 + #8 快捷操作

---

### Feature #4: 🔔 智能告警，语音播报
**标题**: 告警来了？手机震动 + TTS 语音通知
**描述**:
- 接入 Prometheus AlertManager Webhook
- 告警卡片前台展示，后台震动 + 语音播报
- 分级通知（Critical/Warning/Info）
- 一键跳转对应 Pod/Node 详情

**配图**: 截图 #9 告警通知（如果有真实告警截图）

---

### Feature #5: 🌐 多端同步，离线可用
**标题**: PWA + APK 双端，弱网也能用
**描述**:
- Android APK 原生体验（OTA 热更新）
- PWA 版支持 Mac/Win/Linux 桌面安装
- 离线缓存（Service Worker），弱网下也能查历史
- 深色/浅色/跟随系统主题

**配图**: 截图 #10 设置页 或多设备拼图

---

### Feature #6: 🔐 企业级安全
**标题**: JWT 认证 + RBAC 权限，数据安全有保障
**描述**:
- 后端 JWT Token 认证
- 支持 RBAC 角色（admin/operator/viewer）
- HTTPS 加密传输
- Kubeconfig 服务端存储，客户端无明文

**配图**: 登录页 或设置页用户卡片
